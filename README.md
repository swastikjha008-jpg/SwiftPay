## 📖 About

SwiftPay is a modern fintech-inspired digital wallet built with **Next.js**, **Express.js**, **MongoDB**, and **Turborepo**.

It demonstrates how real-world wallet applications handle authentication, balance management, and secure money transfers using **MongoDB replica-set transactions**, ensuring every transfer either completes successfully or rolls back completely.

This project was built to practice scalable full-stack architecture, authentication, backend APIs, and modern frontend development.

---

## ✨ Features

- 🔐 JWT Authentication
- 🔑 Password hashing with bcryptjs
- 💰 Automatic wallet creation
- 💵 Wallet balance management
- ➕ Add money to wallet
- 💸 Secure money transfers
- 🔄 MongoDB Atomic Transactions
- 👥 Search users instantly
- 🧠 Request validation with Zod
- 🎨 Modern responsive UI
- 📦 Monorepo powered by Turborepo

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/SwiftPay.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
